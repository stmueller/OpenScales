Configuration Error

An osc parameter pointing to an .osc study chain file is required.

Example:

chain-runner.html?osc=my_study.osc&pid=P001

URL Parameters

ParameterRequiredDescription
oscYesURL to an .osc study chain file
pidDependsParticipant ID (if required by the .osc)
*NoAny other URL parameters are available to the .osc via ${name} substitution
